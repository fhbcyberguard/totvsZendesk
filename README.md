# Totvs Zendesk
###### _Automação de teste visando a validação da mensagem de erro por e-mail e/ou senha incorretos para processo seletivo com a TOTVS._

### Importante
Para que o teste seja executado com sucesso é necessário instalar as ferramentas listadas abaixo.

## Ferramentas Utilizadas

- [Chrome 107.0.5304.88](https://www.google.com/intl/pt-BR/chrome/)
- [Chrome Driver 107.0.5304.88](https://chromedriver.chromium.org/downloads)
- [Git](https://git-scm.com/downloads)
- [Intellij IDEA 2022.1.1](https://www.jetbrains.com/idea/download/other.html)
- [Java 18](https://www.oracle.com/java/technologies/downloads/#jdk18-windows)

#### Dependências Utilizadas

- [JUnit Jupiter](https://mvnrepository.com/artifact/org.junit.jupiter/junit-jupiter-api)
- [Selenium](https://mvnrepository.com/artifact/org.seleniumhq.selenium/selenium-java)
